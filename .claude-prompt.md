# TRAILERPHIM - AUTO POST MOVIES

Khi user yêu cầu thêm/phim/trailer, tự động làm quy trình sau:

## WORKFLOW
1. Search TMDB theo tên phim
2. Fetch movie details (credits, videos, images)
3. Viết SEO content (content, notable_points, faq)
4. Lưu vào database (Movie + Trailer + Categories)
5. Return SQL/Filament link để confirm

## DATABASE MAP
- Movies: title, slug, description, content, notable_points, faq, poster, release_date, status, duration
- Trailers: youtube_id, title, is_main
- Categories: genres (type='genre'), countries (type='country')

## TMDB API
Base: https://api.themoviedb.org/3
Key: Từ .env (TMDB_API_KEY)

## SEO TEMPLATE
Content: 120-180 chữ, mô tả bối cảnh + nhân vật + xung đột, KHÔNG spoil
Notable Points: 2-3 câu, lý do phim được quan tâm
FAQ: 3 câu Q&A (khi nào chiếu, trailer đã ra chưa, thể loại gì)

## RULES
- Mỗi phim viết HOÀN TOÀN MỚI, KHÔNG copy
- KHÔNG spoil kết thúc
- KHÔNG nhắc "xem full", "tải phim", "download"
- Tiếng Việt chuẩn, giọng trung lập
